# BugCost 💰

> Ferramenta que traduz erros de sistema em prejuízo financeiro real.

## 📝 Descrição

O BugCost ajuda empresas a entenderem o impacto financeiro de falhas técnicas. O sistema analisa logs de erro e cruza esses dados com o faturamento, permitindo visualizar quanto cada falha custa para o negócio em tempo real.

## ✅ Funcionalidades Concluídas

- [x] **Landing Page:** Interface moderna e 100% responsiva.
- [x] **Dashboard:** Painel visual para visualizar os erros e os gastos.
- [x] **Análise de Logs:** Sistema que lê os arquivos de erro automaticamente.
- [x] **Cálculo de Impacto:** Mostra o prejuízo baseado no tempo de erro e faturamento.
- [x] **API Interna:** Rotas configuradas para processar as informações do sistema.

## 🛠️ Tecnologias

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Ícones)
- **Framer Motion** (Animações de interface)

## 🖥️ Pré-requisitos
